# 前进号・周年限定登舰证 H5 开发记录

## 任务
开发一整套纪念船票 H5 交互页面，包含开场动画、主题启动页、信息填写页、3D 船票预览页、邀请分享页。

## 文件
`index.html` — 单文件完整 H5，生产就绪，无需外部依赖。

## 架构：6 场景轮播

### Scene 1 — 夏日海边开场
- 天海渐变背景 + 脉动太阳 + 漂浮云朵 + 海鸥
- 底部三层 SVG 波形动画（不同速度/方向）
- 船 emoji 浮动动画
- "欢迎水手登船" 逐字弹跳动画
- 点击按钮 → 迸发场景

### Scene 2 — 欢迎仪式动画
- Canvas 彩纸粒子迸发（120 粒子，8 色，含重力衰减）
- 金色渐变标题进场动画
- 约 3s 后自动跳转主题页

### Scene 3 — 鎏金主题启动页
- 深色渐变背景 + 浮动金点星空
- 渐变鎏金字体 "前进号・周年限定登舰证"（goldShimmer 动画）
- 信件样式区域展示文案
- 底部固定按钮「开启登舰」+ 底部说明小字

### Scene 4 — 信息填写页
- 固定页眉「「周年限定」前进号」渐变文字
- 三个表单字段（含验证）：
  - **水手昵称**：限制 8 位数字，自动过滤非数字字符
  - **成为水手的日期**：日期选择器，默认今日
  - **选择你的阵营**：🚢海・舰艇 / 🏔️陆・基地 / ✈️空・编队，选中时分别呈现蓝色/绿色/金色边框阴影
- 底部固定生成按钮

### Scene 5 — 3D CSS 卡片翻转（核心）
**正面**：
- 左上「前进号・周年限定」，右上「「2024-2026」」
- 鎏金大字「登舰证」（goldShimmer）+ 灰色英文 "BOARDING PASS"
- 金色航母锚图标
- 金色圆框随机 Q 版海洋生物头像（🐬🐳🐋🦈🐙🦑🦐🦀🐡🐠🐟🐚🪸🐢）
- 昵称
- 航行天数（自动计算从入群日期到今日）
- 舱位编号（A/B/C + 随机 3 位数）
- 船籍编号金色高亮框 "CNS-2026-7541398"
- 阵营 emoji + 文字
- **斜划线金光循环闪烁照射覆盖层动画**（shimmerSweep）

**背面**：
- 「📜 航海日志」标题
- 完整数据面板：昵称 / 入群日期 / 航行天数 / 船籍编号 / 阵营 / 舱位
- 互动语：「水手 {昵称}，欢迎登舰！你的舱位在 B 区 XXX 号，这一路，请多关照 ✨」
- 三色标记点：海蓝⚓ / 陆绿⛰️ / 空金✈️

### Scene 6 — 邀请分享页
- 「登舰证已生成」标题
- 「保存图片」和「分享好友」按钮
- 分享使用 Web Share API，降级为复制链接

## 关键技术实现

| 特性 | 实现方式 |
|------|---------|
| 鎏金字体 | CSS `background-clip: text` + 渐变 + 300% 动画背景 |
| 金光扫描 | `skewX(-25deg)` 渐变长条在 `4s` 周期内左右扫过 |
| 3D 翻转 | CSS `perspective(1200px)` + `rotateY(180deg)` + `preserve-3d` + `backface-visibility` |
| 彩纸迸发 | Canvas 粒子系统（重力、旋转、透明度衰减）|
| 海浪 | SVG 路径重复 + `translateX(-50%)` 动画 |
| 阵营阴影 | 独立 CSS 类 `.selected-sea/land/air`（蓝/绿/金色 + box-shadow）|
| 日期计算 | `Date` 对象差值 → 天 |
